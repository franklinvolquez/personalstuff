AperçuLa description de l'exempleLe code HTML de l'exemple
Affichage par défaut, sans options. 
<object type="application/x-shockwave-flash" data="player_mp3.swf" width="200" height="20">
	<param name="movie" value="player_mp3.swf" />
	<param name="FlashVars" value="mp3=test.mp3" />
</object>
Toutes les options d'affichage sont activées. 
<object type="application/x-shockwave-flash" data="player_mp3.swf" width="200" height="20">
	<param name="movie" value="player_mp3.swf" />
	<param name="FlashVars" value="mp3=test.mp3&amp;showstop=1&amp;showinfo=1" />
</object>
Exemple avec un changement de couleurs. 
<object type="application/x-shockwave-flash" data="player_mp3.swf" width="200" height="20">
	<param name="movie" value="player_mp3.swf" />
	<param name="FlashVars" value="mp3=test.mp3&amp;bgcolor1=ffffff&amp;bgcolor2=cccccc&amp;buttoncolor=999999&amp;buttonovercolor=0&amp;slidercolor1=cccccc&amp;slidercolor2=999999&amp;sliderovercolor=666666&amp;textcolor=0" />
</object>
Exemple avec une image de fond. 
<object type="application/x-shockwave-flash" data="player_mp3.swf" width="200" height="20">
	<param name="movie" value="player_mp3.swf" />
	<param name="FlashVars" value="mp3=test.mp3&amp;skin=skin001.jpg" />
</object>
